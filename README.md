# ESP32-VE-BLYNK-IOT-LE-AKILLI-EK-SULAMA-S-STEM-.
ESP32 ve Blynk platformu kullanılarak geliştirilen bu proje, bitkilerin nem ihtiyacını otonom olarak yönetir. Sistem, tam otomatik (sessiz robot) ve manuel (bildirimli) olmak üzere iki farklı modda çalışır ve harcanan su miktarını bulut tabanlı olarak raporlar.
